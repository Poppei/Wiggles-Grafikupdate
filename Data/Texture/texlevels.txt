# Textur-Levels
# als erstes kommt der Name, danach die Texturauflösungen
# für Minimal, Low, High und Maximal.
# die zu verwendende Auflösung kann angegeben werden als:
# 0 ==  16 == m016
# 1 ==  32 == m032
# 2 ==  64 == m064
# 3 == 128 == m128
# 4 == 256 == m256
# Wenn die entsprechende Stufe nicht gefunden wird, nimmt
# er das, was er für richtig hält. Das bedeutet dann fol-
# gende Suchreihenfolge:
# minimal:   16 ->  32 ->  64 -> 128 -> 256
# low:       32 ->  64 ->  16 -> 128 -> 256
# medium:    64 -> 127 ->  32 -> 256 ->  16
# high:     128 -> 256 ->  64 ->  32 ->  16
# maximum:  256 -> 128 ->  64 ->  32 ->  16
#
# name								minimal 	low		medium		high	maximal
# interface01.tga					32			32		256			256		256
